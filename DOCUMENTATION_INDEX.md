## 📊 Bảng Nhanh - Vai trò & Tính Năng

### Guest
✅ Xem trang chủ | Đăng ký | Đăng nhập
❌ Luyện tập | Tạo bài | Quản lý

### User
✅ Luyện tập | Xem bài | Dashboard | Tiến độ
❌ Tạo bài | Quản lý người dùng

### Teacher
✅ Tạo bài | Quản lý bài | Xem học viên | Phản hồi
❌ Quản lý người dùng | Quản lý hệ thống

### Admin
✅ Quản lý tất cả | Thống kê | Cài đặt | Phân quyền
❌ Luyện tập | Tạo bài (vào vai trò khác)

### AI
✅ Chấm điểm | Phân tích | Phản hồi
❌ Luyện tập | Quản lý

## 🔗 API Endpoints Nhanh

### Auth
```
POST /api/auth/login - Đăng nhập
POST /api/auth/register - Đăng ký
POST /api/auth/logout - Đăng xuất
GET /api/auth/me - Lấy user hiện tại
```

### Lessons
```
GET /api/lessons - Danh sách
GET /api/lessons/:id - Chi tiết
POST /api/lessons - Tạo (Teacher)
PUT /api/lessons/:id - Cập nhật
DELETE /api/lessons/:id - Xóa
```

### Practices
```
POST /api/practices - Nộp bài
GET /api/practices/:id - Kết quả
GET /api/users/:id/practices - Lịch sử
```

### Admin
```
GET /api/admin/users - Danh sách người dùng
GET /api/admin/statistics - Thống kê
PUT /admin/users/:id/role - Thay đổi vai trò
```
